# Utilizzo

## Requisiti

- Docker
- Docker Compose

Assicurati che Docker e Docker Compose siano installati. Puoi verificare l'installazione eseguendo:

```bash
docker --version
docker compose --version
```

Per costruire il container:

```bash

```

Per eseguire il container:

```bash
docker compose run pd-project
```
